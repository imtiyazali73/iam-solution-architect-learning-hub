# IAM Solution Architect Learning Hub

A focused learning workspace for building IAM Solution Architect skills across identity, Zero Trust, IGA, PAM, and cloud IAM.

Live site: [IAM Solution Architect Learning Hub](https://imtiyazali73.github.io/iam-solution-architect-learning-hub/)

Owner mode: [IAM Solution Architect Learning Hub - Owner Sync](https://imtiyazali73.github.io/iam-solution-architect-learning-hub/?owner=1)

## What it includes

- 12-week IAM learning roadmap
- Editable week status, actual hours, and notes
- Progress dashboard
- Focus-week panel
- 25-minute study timer
- Interview practice prompts
- Searchable learning resource library
- Local progress storage in the browser
- Owner-only cloud sync through a private GitHub Gist
- Privacy-friendly visitor analytics through Cloudflare Web Analytics

## GitHub Pages

This project is ready to publish with GitHub Pages from the repository root.

## Cloud Sync

Cloud sync is hidden from the normal shared site link. Owner mode can be enabled locally by opening the site with `?owner=1`.

Cloud sync uses the GitHub Gist API from the browser. Create a GitHub token with Gist access, paste it into the Cloud Sync panel, and press **Connect**. The first push creates a private Gist and saves its ID locally. Use the same token and Gist ID on another device, then press **Pull**.

Your GitHub token is stored only in that browser's local storage.

## Analytics

Visitor counts are tracked with Cloudflare Web Analytics. It provides aggregate traffic insights without tracking individual visitors.
